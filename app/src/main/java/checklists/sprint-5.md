1. Навигация в приложении между экранами работает без падений.
2. Вся логика навигации реализована в классе PlaylistHost.
3. По нажатию на кнопку «Songs» выполняется переход на экран поиска.
4. По нажатию на кнопку «Settings» выполняется переход на экран настроек.
5. При добавлении новых экранов не забывать обновлять код навигации.
